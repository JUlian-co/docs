# Setting up Yi

Yi is proudly [built on open standards](/sdk/yi-vs-others).  As such, you can use Yi through your favorite libraries, notably Wagmi and Viem.

## Installation

If you are going to be using Yi through Wagmi:

:::code-group

```bash [npm]
npm i @build-with-yi/wagmi
```

```bash [yarn]
yarn add @build-with-yi/wagmi
```

```bash [pnpm]
pnpm i @build-with-yi/wagmi
```

```bash [bun]
bun add @build-with-yi/wagmi
```

:::

If you are going to be using Yi through Viem:

:::code-group

```bash [npm]
npm i @build-with-yi/viem
```

```bash [yarn]
yarn add @build-with-yi/viem
```

```bash [pnpm]
pnpm i @build-with-yi/viem
```

```bash [bun]
bun add @build-with-yi/viem
```

:::
